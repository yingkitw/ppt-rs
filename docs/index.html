ppt-rs

Rust A Rust library for generating PowerPoint (PPTX) files.

Quick Start

use ppt_rs::prelude::*;

let slides = vec![
    SlideContent::new("Hello World")
        .layout(SlideLayout::CenteredTitle)
        .title_bold(true),
];
let pptx = create_pptx_with_content("My Presentation", slides)?;
std::fs::write("output.pptx", pptx)?;

Key Features

  • 40+ shape types with fills, lines, text
  • Tables, charts, images, connectors
  • Flexible Dimension API (EMU, inches, cm, pt, ratio, percent)
  • Fluent builder pattern with chaining
  • Bullet styles, text formatting, themes
  • Slide transitions, animations, notes
  • Valid PPTX output compatible with PowerPoint
